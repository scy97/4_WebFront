display 속성

        - block : 화면을 수직 분할 + width, height 사용 가능
        - inline : 화ㅕㅁㄴ을 수평 분할 + width, height 사용 불가능
        - inline + block : inline의 수평 분할 + block의 크기 조절
        - none : 화면에 요소가 표시되지 않으나 존재는 하고 있는 상태
        - flex : 요소의 정렬되는 방향, 간격을 유연하게 처리하는 형식
